Student Portfolio Website
A clean and modern **student portfolio website** built as a **first semester project** using HTML and CSS.  
This project focuses on learning the basics of web development, responsive design, and UI styling.

##  About the Project

This portfolio website is designed to showcase:
- Student introduction
- Skills and progress
- Academic projects
- Contact information
The goal of this project is to practice **HTML structure**, **CSS layouts**, and basic **frontend design principles**.

##  Features
- Responsive layout (mobile-friendly)
- Fixed navigation bar
- Smooth scrolling navigation
- Skills section with progress bars
- Project showcase cards
- Contact form (demo)
- Clean and modern UI design

##  Technologies Used
- **HTML5**
- **CSS3**
- **Basic JavaScript** (for smooth scrolling & demo alert)
- **Google Fonts (Poppins)**
